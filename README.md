# ✈️ TravelClub CRUD Application

Spring Boot로 구현한 **여행 동아리(Travel Club) CRUD 웹 애플리케이션**입니다.  
REST API 기반으로 동아리 생성, 조회, 수정, 삭제 기능을 제공합니다.

## 💾 H2 콘솔
- URL: `http://localhost:8090/h2-console`
- JDBC URL: `jdbc:h2:mem:test`

## 🎯 학습 포인트
- Spring Boot MVC 아키텍처 이해
- JPA 기반 CRUD 구현
- 계층형 구조(Controller-Service-Store) 설계
